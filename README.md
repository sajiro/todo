

# Reactjs Todo App

## Getting Started

### 1. Clone the Repository

```sh
git clone git@github.com:sajiro/todo.git
```

### 2. Install Dependencies

```sh
npm install
```

### 3. Run the Development Server

```sh
npm run dev
```

### 4. Open your browser and navigate to

```sh
http://localhost:5173
```
